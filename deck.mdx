# Konfiguration

```yaml

language: node_js
node_js: stable
install:
  - npm ci
script:
  - npm run build
deploy:
  provider: pages
  local-dir: ./dist
  skip-cleanup: true
  github-token: $GITHUB_TOKEN
  on:
    branch: master
```
